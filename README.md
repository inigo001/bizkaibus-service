# BizkaibusService

Prueba de concepto para pasar los servicios de la API de **Bizkaibus** a algo manejable con Javascript. Todavía inútil y muy poco funcional, probablemente de manera perpetua.

## Métodos disponibles

* `getFromTo`
* `getHorario`
* `getPdf`
* `getItinerario`
* `getParadasTown`
* `getPasoParada`
* `getLineInfo`
* `getVehicles`
* `getLines`
* `getBusInfo`
* `getTowns`
